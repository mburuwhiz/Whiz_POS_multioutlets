## <a name="quick-start"></a>🚀 Quick Start

### Prerequisites
- Node.js 18+
- MongoDB 6+
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/whizpoint-pos.git
   cd whizpoint-pos/website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. **Start MongoDB**
   ```bash
   # Make sure MongoDB is running locally or use MongoDB Atlas
   ```

5. **Run the development server**
   ```bash
   npm run dev
   ```

6. **Open your browser**
   Visit [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
# Build for production
npm run build

# Start production server
npm start
```

## 🔧 Configuration

### Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `MONGO_URI` | MongoDB connection string | `mongodb://localhost:27017/whizpos` |
| `TURSO_DATABASE_URL` | Turso database URL (optional) | - |
| `TURSO_AUTH_TOKEN` | Turso authentication token (optional) | - |
| `API_AUTH_KEY` | API authentication key | - |
| `ADMIN_EMAIL` | Global admin email | `admin@pos.whizpoint.app` |
| `ADMIN_PASSWORD` | Global admin password | - |
| `SMTP_HOST` | SMTP server host | `smtp-relay.brevo.com` |
| `SMTP_PORT` | SMTP server port | `587` |
| `SMTP_SECURE` | Use SSL for SMTP | `false` |
| `SMTP_USER` | SMTP username | - |
| `SMTP_PASS` | SMTP password | - |
| `BREVO_API_KEY` | Brevo (Sendinblue) API key (optional) | - |
| `EMAIL_FROM` | From email address | `info@pos.whizpoint.app` |
| `REPLY_TO` | Reply-to email address | `admin@whizpoint.app` |
| `SESSION_DURATION` | Session expiration | `5m` |
| `NEXT_PUBLIC_APP_URL` | Public app URL | - |

### Session Duration Units
- `s` = seconds
- `m` = minutes
- `h` = hours
- `d` = days
- `mo` = months

Examples: `15m`, `1h`, `7d`, `1mo`
